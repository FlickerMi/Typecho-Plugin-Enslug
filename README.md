# Enslug for Typecho

**Enslug** 是一款 Typecho 博客插件，能够自动将中文文章标题翻译为英文缩略名（Slug），让文章链接更加美观且有利于 SEO。

## 功能特性

- 自动将中文文章（或独立页面）标题翻译为英文，生成对搜索引擎友好的 URL 缩略名。
- 接入最新版百度翻译 API，保证翻译的准确性。
- 后台支持自定义配置百度翻译 API 的 **APP ID** 和 **密钥 (Key)**。
- 修改已有文章标题时，提供更新确认提示，防止原有的链接缩略名被意外覆盖。
- 完美兼容并支持 HTTPS 站点。

## 安装与使用

1. 下载本插件压缩包，解压后将文件夹重命名为 `Enslug`。
2. 将整个 `Enslug` 文件夹上传至 Typecho 站点的 `usr/plugins/` 目录下。
3. 登录 Typecho 后台，在 **控制台 > 插件** 列表中找到 `Enslug` 并点击 **启用**。
4. 启用后，点击插件的 **设置** 按钮：
   - 填入您自己申请的 **百度翻译 API** 的 `APP ID` 和 `密钥 (Key)`。
   - 如果您还没有百度翻译 API 密钥，可以通过 [百度翻译开放平台](http://api.fanyi.baidu.com/api/trans/product/desktop?req=developer) 免费注册并申请开发者接口。
